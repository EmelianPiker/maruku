Maruku: a Markdown-superset interpreter

Maruku is a Markdown interpreter written in Ruby.

The official repository and issue tracker are on GitHub: http://github.com/bhollis/maruku

Authors: Maruku was started by Andrea Censi. The current maintaner is Benjamin Hollis. Many other people contributed patches; a partial list is in the AUTHORS file in the source distribution.

The name of the game: Maruku is the romaji transliteration of the katakana transliteration (マルク) of “Mark”, the first word in Markdown. I chose this name because Ruby is Japanese, and also the syllable “ru” appears in Maruku.
